Windows 11 admitirá la reversión a Windows 10, pero no por mucho tiempo.

  • Alpha Blog

¿Cómo sabrá si vale la pena usar Windows 11? Hay un programa de vista previa para Windows 11, pero a las compilaciones de vista previa todavía les faltan algunos elementos de la versión final. No tiene que meterse con las compilaciones de Insiders en absoluto: puede instalar la versión final cuando esté disponible y probarla.

Esta noticia llega a través de un PDF que Microsoft ha proporcionado a los fabricantes de PC . Es un formato de preguntas frecuentes, y entre las diversas consultas redundantes se encuentra esta joya: "¿Puedo volver a Windows 10 después de actualizar si no me gusta Windows 11?" La respuesta es un sí rotundo … durante 10 días. Tendrá ese tiempo para decidir volver a Windows 10. Espere más y estará bloqueado en Windows 11 a menos que vuelva a formatear su sistema.

Si opta por volver a Windows 10, podrá hacerlo con solo unos pocos clics. La opción está en Configuración> Actualización y seguridad> Recuperación. Simplemente seleccione la versión anterior de Windows 10 y volverá a estar donde estaba con todas sus aplicaciones y datos intactos. En el pasado, la actualización a una nueva versión de Windows, incluso una de las compilaciones de funciones bianuales, dejaba en blanco los puntos de restauración del sistema anterior.

Si ni siquiera quiere considerar Windows 11, esa también es una opción compatible. Microsoft admitirá Windows 10 con actualizaciones hasta octubre de 2025. Según el éxito de Windows 11, es posible que incluso tenga que extender el soporte de Windows 10 un poco más.

Windows 11 realiza una serie de cambios importantes en la interfaz de usuario, incluido un nuevo menú de inicio sin mosaicos en vivo. Tendrá un acceso más rápido a las aplicaciones ancladas y los archivos recientes, pero los fanáticos de Cortana (si existe tal cosa) se sentirán decepcionados al ver que el asistente de voz de Microsoft está metido en una aplicación apartada. Las esquinas redondeadas también abundan en el nuevo software. La Tienda Windows también recibirá una actualización importante, con soporte para más tipos de aplicaciones y métodos de pago
